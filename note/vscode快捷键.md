在使用 Visual Studio Code (VS Code) 时，有许多实用的快捷键可以提高你的编码效率。以下是一些常用的快捷键推荐：

1. **打开文件**:
   - `Ctrl + P`: 快速打开文件。
   - `Ctrl + R`: 快速打开文件或命令。

2. **编辑操作**:
   - `Ctrl + X`: 剪切选中的文本。
   - `Ctrl + C`: 复制选中的文本。
   - `Ctrl + V`: 粘贴文本。
   - `Ctrl + Z`: 撤销操作。
   - `Ctrl + Y`: 重做操作。
   - `Ctrl + U`: 将选中的文本转换为小写。
   - `Ctrl + Shift + U`: 将选中的文本转换为大写。

3. **导航**:
   - `Ctrl + G`: 跳转到指定行号。
   - `Ctrl + Shift + O`: 快速打开符号（变量、函数等）。
   - `F12`: 跳转到定义。
   - `Alt + F12`: 查看定义。
   - `Ctrl + Shift + F12`: 查看所有引用。

4. **多光标编辑**:
   - `Alt + Click`: 在当前光标位置添加一个光标。
   - `Ctrl + Alt + Arrow`: 向指定方向添加多个光标。

5. **代码折叠**:
   - `Ctrl + Shift + [`: 折叠当前代码块。
   - `Ctrl + Shift + ]`: 展开当前代码块。
   - `Ctrl + K Ctrl + [`: 折叠所有代码块。
   - `Ctrl + K Ctrl + ]`: 展开所有代码块。

6. **代码格式化**:
   - `Shift + Alt + F`: 格式化当前文件。
   - `Ctrl + Shift + I`: 格式化选中的代码。

7. **代码重构**:
   - `F2`: 重命名符号。
   - `Ctrl + .`: 快速修复（显示可快速修复的错误和警告）。

8. **搜索和替换**:
   - `Ctrl + F`: 在文件中查找。
   - `Ctrl + H`: 替换文本。
   - `Ctrl + Shift + L`: 替换所有匹配项。

9. **窗口管理**:
   - `Ctrl + \`: 切换侧边栏的显示/隐藏。
   - `Ctrl + Shift + E`: 打开资源管理器。
   - `Ctrl + Shift + P`: 打开命令面板。

10. **扩展管理**:
    - `Ctrl + Shift + X`: 打开扩展视图。

11. **终端操作**:
    - `Ctrl + `` `: 打开/切换终端。
    - `Ctrl + Shift + ` `: 创建新终端。

12. **调试**:
    - `F5`: 启动调试。
    - `F9`: 切换断点。
    - `F10`: 单步跳过。
    - `F11`: 单步进入。

这些快捷键可以帮助你更快地导航、编辑和调试代码，从而提高工作效率。
